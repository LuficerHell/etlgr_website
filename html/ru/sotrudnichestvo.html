Переводы

Не стесняйтесь добавлять свой файл перевода в папку app/dict/langs. Больше вам не нужно ничего делать. Файл перевода - это единственная вещь, которая требуется, чтобы добавить новый язык

Схема

Формат файла очень простой:

# -*- coding: utf-8 -*-

Вы можете скопировать файл EN.py, переименовать его, используя любое уникальное имя, а затем перевести все фразы

Тестирование

Мы используем py.test, поэтому, чтобы запустить тесты, выполните следующую команду:

py.test tests

Тестирование

  • Сделайте форк этого репозитория
  • Сделайте все необходимые модификации
  • Запустите тесты, убедитесь, что всё в порядке
  • Сделайте сommit ваших изменений
  • Сделайте pull request
